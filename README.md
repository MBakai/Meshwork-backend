# Meshwork

1. clonar proyecto.
2. __yarn install__
3. clonar archivo __.env.template__ y renombrarlo a __.env__
4. cambiar variables de entorno.
5. crear y levantar base de datos con docker:
````
docker-compose up -d
````
6. ejecutar el proyecto en modo desarrollador.
````
yarn start:dev
````
